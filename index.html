4 is Cosmic!

  1. Say a non-negative number
  2. Determine how the subsequent numbers are calculated.
  3. Repeat step 1 as necessary.

Note: If requested, allow permission to use the microphone.
Headphones are recommended.

Listening...

Reveal Answer The next number is calculated by the length of the number provided in word form. For example, the number 5 results in "five", which is a length of 4. "four" ends the sequence.
